# README SETUP

## New Project

`npx nuxi@latest init <project-name>`

`cd <project-name>`

`yarn / npm`

`yarn dev / npm run dev`

## Alias

in Nuxt3 you can use "~" to be alias as root

ref: <https://nuxt.com/docs/guide/going-further/layers#basic-example>

## Extension

-Vue Volar extension Pack
-Vue VSCode Snippets
